PokemonrandomizerAC

An Android app for randomizing supported Pokemon ROMs directly on an Android phone, tablet, or handheld console.

Download APK Original Randomizer ZX

What It Does

Randomize On Device

Pick a ROM, choose an output file, configure settings, and create a randomized ROM without a desktop PC.

Controller Friendly

The app is built for handheld Android consoles and can be navigated with a controller.

Detailed Settings

Use the settings menu to adjust randomization options for Pokemon, moves, trainers, items, evolutions, and more.

Install

  1. Download the APK using the button above.
  2. Copy it to your Android device if you downloaded it on another computer.
  3. Open the APK on Android and allow installation from your browser or file manager if prompted.
  4. Launch PokemonrandomizerAC and select your legally owned ROM file.

Requirements

  • Android 8.0 or newer.
  • A legally owned ROM file for a supported Pokemon game.
  • Enough storage space for the input ROM and randomized output ROM.
